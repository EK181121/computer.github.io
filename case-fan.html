機殼風扇 (Case Fan)

System Cooling - 電腦的通風系統

基本定義

機殼風扇安裝在機殼的前方、後方或頂部，負責主動推動空氣流動，將外界冷空氣吸入，並將機殼內的熱廢氣排出。

功能說明

  • 進氣 (Intake)： 通常位於前方或底部，吸入新鮮冷空氣。
  • 排氣 (Exhaust)： 通常位於後方或頂部，將 CPU 和顯卡產生的熱氣抽出。
  • 正/負壓差： 進氣大於排氣為正壓（防塵較佳），排氣大於進氣為負壓（散熱稍好但易積塵）。

系統角色與交互關係

輔助 CPU 與 GPU

即使 CPU 散熱器再強，如果機殼內部都是熱空氣，效果也會大打折扣。機殼風扇確保了「環境溫度」足夠低。
