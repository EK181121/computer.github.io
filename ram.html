記憶體 (RAM)

Random Access Memory - 電腦的短期記憶區

基本定義

RAM (隨機存取記憶體) 是電腦的短期記憶區。它存放 CPU 當前正在使用的作業系統、應用程式和資料，以便 CPU 能以極快的速度存取。

功能說明

  • 高速讀寫： 速度比硬碟 (SSD/HDD) 快上百倍。
  • 揮發性儲存： RAM 需要電力來維持資料，一旦關機或斷電，資料就會消失。
  • 多工處理基礎： 容量越大，您可以同時開啟的程式或分頁就越多。

系統角色與交互關係

硬碟與 CPU 的橋樑

硬碟像是一個巨大的倉庫（慢但大），CPU 是工匠（快）。RAM 就是工作桌，先把東西從倉庫搬到桌上，工匠才能快速工作。
