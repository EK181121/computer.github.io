中央處理器 (CPU)

Central Processing Unit - 電腦的大腦

基本定義

CPU 是電腦中負責解釋和執行指令的核心元件。就像人類的大腦一樣，它負責處理來自軟硬體的所有請求，進行算術、邏輯、控制和輸入/輸出 (I/O) 操作。

功能說明

  • 指令週期 (Instruction Cycle)： 執行擷取 (Fetch)、解碼 (Decode)、執行 (Execute) 的循環。
  • 算術邏輯運算： 透過 ALU (Arithmetic Logic Unit) 進行加減乘除與邏輯判斷。
  • 控制協調： 指揮其他硬體元件（如記憶體、硬碟）何時傳輸資料。

系統角色與交互關係

CPU 位於資料處理的中心：

與 RAM

CPU 從 RAM 讀取指令與資料。RAM 的速度與容量直接影響 CPU 的等待時間。

與 GPU

CPU 負責計算物理與邏輯，然後指揮 GPU 繪製畫面。如果是玩遊戲，CPU 就像導演，GPU 則是畫師。
