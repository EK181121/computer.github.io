顯示卡 (GPU)

Graphics Processing Unit - 電腦的眼睛

基本定義

GPU (圖形處理單元) 是一種專門設計用於快速處理和修改記憶體的電子電路，以加速圖像在顯示器上的建立。現代獨立顯示卡通常包含自己的記憶體 (VRAM)。

功能說明

  • 3D 渲染： 處理多邊形、紋理映射、光影計算 (如 Ray Tracing)。
  • 平行運算： 擁有數千個核心，極其適合處理大量重複的並行任務（如 AI 運算、影片解碼）。
  • 影像輸出： 將數位訊號轉換為螢幕可顯示的影像。

系統角色與交互關係

CPU 的畫師

CPU 告訴 GPU「這裡有個場景，有這些物件」，GPU 負責算出每個像素的顏色。

VRAM 的重要性

GPU 需要在 VRAM 中暫存大量的紋理貼圖。高解析度 (4K) 遊戲需要極大的 VRAM 容量。
